WYSIWYG + Source + Split

Three viewing modes with Milkdown 7 (ProseMirror). Full GFM support, KaTeX math, Mermaid diagrams.

◇

Sandboxed AI

OpenAI, Claude, Gemini, Ollama — all through Wasmtime. Every tool is isolated with capability-based security.

⊞

Document Ingestion

PDF, DOCX, PPTX, XLSX, EPUB, YouTube, Wikipedia, RSS → Markdown via Python MarkItDown.

↔

MCP Integration

First-class MCP client with stdio and HTTP transports. Connect GitHub, databases, and custom tools.

♯

Git Version Control

Auto-commit on every save. Browse history, view diffs, push to GitHub — all from within the editor.

🔒

Private by Design

No intermediary servers. Your prompts go directly to your chosen provider. Zero telemetry. Offline-first.
